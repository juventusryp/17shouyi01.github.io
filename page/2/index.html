入学感想-李亚楠

发表于 2018-01-28 | 分类于 那些想说的话

  在丹桂飘香、丰收的金秋五月，我怀揣梦想，满怀期待地踏入了大学校园。一段崭新的学习生活即将开始。
  记得刚踏进校园时，我拎着大包小包，迷茫地走着，摸不清方向，是热心的学姐、学长们对我关怀备至。他们热情地从我手中拿走包裹，帮我提着，带我报道、送我到宿舍。突然就觉得心里有一股暖流在涌动着，他们让我看到了同学之间那种真挚又可贵的爱。
红彤彤的烈日下，我迎来了期待已久的大学军训：挥洒汗水，斗志昂扬，我带着流血流汗不流泪的心，度过了十二个旭日东升和夕阳西下。军训留给我的是坚定的毅力，是严于律己，树立规范意识，是面对挫折，苦难咬牙坚持，迎难而上。

入学感想-张尚

发表于 2018-01-28 | 分类于 那些想说的话

  我理想中的大学应该是一个有活力，充满激情向上的学校。可以不是特别好的学校，但是里面的同学们一定要有积极向上的态度来对待学习和生活。我希望它有许多的有趣社团，并不是那种只一个星期开一次例会的社团。是那种真正能让我们感到开心和学习到知识的社团。还有学校组织的活动要不能只流于形式，一定要真正的组织起同学们参与。我还希望里面有许多平易近人的老师，他们不仅资历深厚，而且风趣幽默。我还希望有一些乐于助人喜欢帮助我们的学长学姐们。哈哈，这就是我理想中的大学。

欢度元旦---李克勤 潘世登

发表于 2018-01-03 | 分类于 班级活动

  生活不止眼前的狼狈，还有美好的昨天和未知的明天。

  眨眼间已是春光明媚，又是一年新学期，新生活，新起点。而过往经历的种种似乎还在昨天。不仅是初入大学报道的场景，还有和朋友们一起度过的难忘的青马培训活动，以及犹在昨天一般的元旦晚会等……而其中最为难忘的大概是去年初入大学时举行的迎新晚会。

  作为一个有幸参加的成员，当初和学长学姐以及同学们相识的场景还历历在目。从在班级群里看到征集节目时的心动，到在军训时的心不在焉，再到海选时的忐忑不安，以及训练过节目时的疲惫难过，直至最后正式演出时的开心喜悦… 其中种种，令人难忘，无需与外人道 已然铭记于心。
