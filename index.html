辩论赛有感---吴忧

发表于 2018-04-28 | 分类于 院级比赛

  我想大学和高中带给我最直观的区别，就是各种活动、比赛、讲座，而在这场辩论赛之前，所有的自愿活动和比赛都是和我无缘的。参加辩论赛的初衷也是一时玩笑，谁也没有想到能走到最后。

  因为是第一次打比赛，所以现在回想起来最开始的晋级赛准备的特别不充分，我们四个人磨合商量，反驳防守的方式很奇怪，两两一组站相对的辩题互相反驳，过程十分激烈，从理论发展到抬杠甚至发展到人身攻击，令人意外的是这样效果很好，大家为了拼命diss对方，寻找到很多独特的观点，在这个过程中也都找到了很好的定位，而赛场上对方所提到的点也基本都在我们所掌控的范围内，有趣的是每当有人提出很尖锐的对方论点，我们经过讨论找到破题的方法，就很自傲的感叹，赢了赢了。但是当对方提到的点我们都有查过资料时，感受到的并不是幸运，是我们对比赛下了苦功夫，翻阅的资料远大于比赛时所提及的，赵玉清赛场上精彩简短的反驳，背后是激烈争执中反复演练的回答，王阳枯燥的开篇立论，是改了多次，怕对方捉住对方其中任何一句有偏颇的话。

辩论赛的妙处---王阳

发表于 2018-04-28 | 分类于 院级比赛

  长达一个月的辩论赛终于结束了，每次出了新辩题我们都要激烈争论好久，一番狂轰乱炸之后，我们都会想出不同的点，去反驳对方，这个时候是最开心的时候。其实我个人平时不喜欢与人进行争辩, 因为我觉得很多事情都是仁者见仁智者见智的, 没有什么对错, 非要分出是非我觉得那是无聊。

  我从来没参加过辩论赛, 所以在辩论的过程中出现了很多错误，比如不熟悉与人辩论的技巧, 讲话不流利, 词不达意, 这些都是我自己 的不足之处,这次的辩论赛对我来说是一次挑战,也是一次难得的历练。在此次辩论赛中，我总结出了一些心得体会：第一，团队精神。我们组有四名成员，我们分工各不相同，任务有轻有重，但是我们大家各司其职，在准备的过程中，我们充当正反角色进行辩论反驳，这使得我们不断碰撞出新的火花，这时候，打辩论赛的快感喷涌而出，一个月内不连续的辩论赛，我们辩手互相理解互相帮助，互相学习，让我们收获很多，在思辨能力方面得到了锻炼。

发表于 2018-01-29 | 分类于 那些想说的话

  转眼之间，已经上了大学。经过了半个学期的学习我发现真正的大学和我想象中的有些不一样，人们都说高中好好学习吧，到了大学你就解脱了。到现在我才明白原来我们从高中这个火坑跳到了另外一个火坑里。
  曾经我认为大学是一个谈情说爱的地方。那有没有教导处的监管，没有大量的课后作业，也没有成摞成摞的试卷。有的只是丰富的课余时光，合得来的室友，关系融洽的小伙伴，各种有趣的活动，美丽的校园景色以及成对成对甜蜜的情侣。有课余时间就在图书馆里翻找资料，和对象一起在学校附近有名的地方游玩，或者和室友时不时的一起去KTV唱歌，去网吧开黑。做自己喜欢做的事，没有父母唠叨不休的教导，可以说是真正的放飞自我。

青春不负自己-孙亚敏

发表于 2018-01-29 | 分类于 那些想说的话

  在进入大学之前，老师和家长都会对我们说大学是自由的天堂。当时我对这句话是深信不疑的。想着只要考上了大学，我就再也不用每天背化学公式，写英语单词了，不用每天面临着数不尽的试题，每天做题做到十一二点了。于是就开始为考大学而努力的学习。我们高三的班主任是一个刚毕业的大学生，没有其他资深的老师那么有经验。为了让我们更加的努力学习，就时不时的跟我们讲她上大学时的一些趣事，她告诉我们：她在洛阳上的大学，洛阳有名的地方她都去腻了，周末的时候和朋友逛逛商场啊，买买衣服啊，到其他城市找老同学玩啊，每天两三节课，业余时间很多，总之，上了大学特别轻松，特别自由。确实每次听她讲后，我都特别想尽快的进入大学。十二年的寒窗苦读，十二载的披荆斩棘，承载着太多的梦想和追求，欢乐与痛苦，经历过千军万马过独木桥的汹涌之势，怀着期盼的心，踏着希望的音符，大家来到这个梦想起飞的地方“河南牧业经济学院”。现在我们已经经过了大一的一个学期，此时此刻不知道大家的心情是否和我一样，有了半年的大学生活，我们都相信了：命运并非上天的安排，命运是掌握在自己的手中，要获得幸福，拥有辉煌，就得去耕耘去锻造，就得左右自己。
