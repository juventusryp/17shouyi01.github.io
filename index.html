班级运动会

发表于 2018-04-04 | 分类于 班级活动

  时间太瘦，指缝太宽。我们的第一学期已经恍恍而过，我觉得刚入大学的第一个大型活动应该就是我们的秋季运动会了。运动会里最靓丽的风景应是运动员们奋力体育的留下的恢恢汗水；应是同学们呐喊助威用热情筑建的战鼓；应是后勤人员来回奔波的身影；应是老师倾心的鼓励和支持。

  首先我们应该致敬那么一群人，他们是赛场上的战士，我们的运动员们。也许运动会对我们来说只有两天，算是短暂吧，但我们的运动员们却早早熟悉了凌晨六点钟的龙子湖。他们在奔跑中迎接朝阳，在训练里体验晨曦。他们代表校区、代表班级，在日复一日的刻苦训练中他们一步步地接近目标、接近荣誉。
班级运动员：李江鹏、李秋平

  另一些在运动会还未开始就已经准备的人就是我们的方队人员，在不知多少的夜里大家进行着一遍又一遍的方阵练习。或许他们不熟悉龙子湖六点钟的早晨，但他们却是清楚牧经九点的夜景。他们的努力铸就院系的风采，追逐着属于他们的荣耀。每个练习的夜晚都是值得纪念的时光。
方队成员：孙亚敏、王晨、陈俭颖、王梦军、袁崧晨、季梦莉、李亚楠、李如月、刘亚锦、李连成、常鹏昌、许红村、马晨洋、邱宇烈、袁靖松、张博、巩云龙

  记得中招时候为了准备体育考试，我们一次一次累倒在一千米的训练中，经常认为一千米是那么遥不可及。后来的我知道了还有五千米这种逆天项目，但也没有过触碰。没有那份勇气的我遗憾没有感受过那种酣畅淋漓的挥洒汗水、释放自我。但每一个参加运动会的同学都值得让人尊敬。毕竟有种坚持让人神往。

  接上话，还说方队还有其他后勤人员。一群奉献的人。除了负责方队的训练和展示，他们还是运动会的主要拉拉队成员。看台上我见到的是一次次歇斯底里的呼喊、感到的是比阳光还要灼热的情绪，是大家的无数的鼓励和心底的支持。我不清楚到底有多少人沙哑了嗓子，但我却真切的感受到大家无私地奉献。
拉拉队：方队成员、后勤人员

  最后，是我们的后勤人员。远在运动员训练之前，阮老师就和体育部的同学进行了运动会的策划和各个细节的审核。学生会各部门也都投入了运动会的准备。在每天同学们训练之前都会有学生会的同学轮流为训练的同学煮鸡蛋、牛奶，还有体育部的同学对大家的训练进行监督、指导。运动会时每个后勤人员负责一个运动员，给运动员们送水、拿衣服……大家都尽力做到细节。阮老师身兼数职，在参加运动会的同时还特地为拉拉队的同学买了金嗓子。每一个后勤的老师和同学都早去晚归，没有饭点、没有规律全身心的为运动会投入。
后勤人员：阮老师、张博、张尚、孙亚敏、毛翔、巩云龙

  越努力越优秀，认真付出的人不会被辜负。大家虽然没有取得最好的名次，但还是获得了应有的荣誉。应该没有言语能描绘大家的喜悦和兴奋。那天回去的路延伸到了黑夜，也许是因为堵车、也许是因为疲累，回去的车是静的、人也是静的，静在深深的夜里。
阳光洒下伴着金叶，激情都留在看台还有塑胶场地。运动会虽然结束，热血、努力还有每一份的善良都是记忆犹新。

编写人：张尚、巩云龙
活动日期：2017年10月27—28

广播操

发表于 2018-04-03 | 分类于 班级活动

  为了让我们得大学过的多姿多彩，我们学校举办的许多活动，比如广播体操比赛、运动会等等，这些活动使我们的大学生活不那么的枯燥乏味，有了一些味道。最让我印象深刻的是广播体操比赛，为什么对广播体操印象深刻呢？首先，我参加可这项活动，我也在那些天留下了辛苦的汗水；其次，这是我参加的第一个大学活动，我也为之付出了自己的努力。

  虽然广播体操，已经过去了很久，但在记忆中还能找到当时的场景。

  那天，在班级告知学校将会以院系为单位开展广播体操比赛活动时，我和宿舍的伙伴们都积极的去参加这个活动，不久我们就建立了一个“广播体操比赛群”用于发通知，开始的几天训练都是在早晨，由于当时的天气也非常冷的，有些人到了训练的点，却迟迟不来，起初的训练是非常困难的， 虽然广播体操在中学时代也做过，但现在我们早已忘的差不多了，所以学长就每一节的、每一个动作的去教我们，因为人数较多，一个人的动作出错就会影响到整个队伍，有时一个动作做的不好，学长就会让我们一直保持这个动作，然后一个一个的去纠正我们的错误，期间也对我们发过脾气，随着比赛的推进，对人员的要求，我们庞大的队伍不得不淘汰人，训练也有每天的早晨改到中午和下晚自习后，天气越来越冷，还好我们都坚持下来了，在比赛的前天晚上我们的训练照常继续，不过这一次我们更加认真，因为我们知道比赛明天就要开始了，相比之前，我们有了很大的进步，动作都能跟得上节奏了，不在是那么的散漫了，那晚训练之后，我们就回到了宿舍，等待着第二天的比赛，第二天早早我们都换好统一的服装，等待着比赛的最后一刻，我们不慌不慢的排好队形，跟着广播体操的节奏，向观众展现我们的成果，虽然我不知道我们，最后有没有赢得名次？但真的算是体会到“台上一分钟，台下十年功”那种过程。没想到还会给我们发证书，非常感谢，我能有此次的，体验。

  时间既逝，转眼间进入下学期，但愿在以后会有更好的明天。

编写人：袁靖松，邱宇烈
活动日期：2017年10月21日

军训

发表于 2018-04-03 | 分类于 班级活动

  军训，是我们大一新生进入大学的第一节课，也是大学中最严格的一节课。有的同学听到军训就会想到暴晒的阳光、黝黑的皮肤、苛刻的训练、还有黄教官那不怎么严肃的表情。的确，军训时期没有正常学习生活过的舒服，但它却让我们难得体会到了后备军或其他军官在壮大国防力量时所受的辛苦和劳累。况且我们做的那些训练对于教官们来说“so easy”，根本不算什么。

  军训时期，我还记得教官们离开的那一天。精心准备的表演是他们留给我们的礼物，是如此的惊心动魄啊。有的时候，看着自己的教官被另外一个教官当作敌人摔打时，心里挺不舒服的，那怕自己的教官平时对同学们多么的苛刻，多么的严厉，也会皱眉心痛。其实我们都知道，教官对我们挺好的，只不过因为他们有任务在身，不得不严厉对待学生，不然他们将会受到更严厉的惩罚。我曾经亲眼看到过在学生们解散去吃饭的时候，有二十多位教官不知何故被集体受罚；还有一次，三个教官在训练学生期间被罚下蹲一个小时。没有人会听他们解释，也没人能体会他们那时复杂的感受。所以在教官们离开的那一刻，多么能理解为什么有些同学都哭了，并且还抱怨着学校为什么不让我们送送他们，送送我们那可爱的教官们。

  一晃而过，军训的时光早远去，教官们也慢慢的淡出我们的记忆，但军训的那这事永远是我们不会忘记的话题。

  满怀热血的我们，即将迎来期盼已久的12天军训生活，谁也不知道，在接下来的几天中会发生什么，那就让我们拭目以待吧！

  第一天，在列日映衬下的我们，一个个都显得那样顽强，男生在军姿的调整下，更加魁梧；女生气质更加。我印象最深的是常鹏昌（海拔最高），在队伍中比较显眼，教官走到他面前时，只见他头微低，瞳孔向下滚动，嘴角向上扬起，露出他那满排的牙齿，好一个灿烂的笑容。（对了，他还是个吉他高手。）教官在常鹏昌的衬托下，显得那样小巧玲珑。班长孙亚敏，有一头乌黑亮丽的秀发，已过腰，由于军训需要每天戴脱军帽，她总是将头发高高盘起，可见她，对待事情一丝不苟，认真负责。这就是我们的两名军训负责人。

  在训练期间，郎建华学姐和刘振学长，会时不时的来查岗，学姐很萌，亲切温柔，学长面无表情时，“生人勿进”，千万不要说话，当然其余时间，说什么都可以啦。

  我性格大大咧咧，自来熟，所以即使来17兽医01这个大家庭没两天，就已经认识了三分之二的人：可爱的小天使，“百灵鸟”样的王晨，关爱他人的钱梦涵，开心果赵玉青，女神吴优，Fishion王阳，淑女李晓（大爱四人帮！形影不离）。
萌萌哒王梦攀，小可爱刘亚锦，班级第一张佳文，两名才女：弹古筝的赵松然、舞姿优美的潘世登，认真努力的陆婷，帅气的杨静，迷人的杨梦丹（在军训时，我们到姚桥一游，很开心第一次和朋友出校门玩。）小美女李如月，大美女王蓉萱，爱笑的王孟军，矜持的陈俭颖，沉着的刘慧芳，乖巧的宋会娜，让我一直分不清的李秋平和何璐冰（不过现在分清了。）腼腆的于雅琳、邓照新、胡霞、吴瑞瑞，想说我们班应该是最富有的，因为有个大金子（金世煊），不管说什么都不会尬场，因为有季梦莉。男同胞们就不说。这就是我在军训期间认识的你们，这个大家庭里面的各位美女。

  咳咳！最最最重要的！618,6个美妞集合啦，宠物1班，大眼睛，樱桃嘴香香；宠物2班，完美到无可挑剔的平平；我们班最热情，乐于助人的“618头头”翟翟；最善解人意，高冷范的也子；我们寝室最会保养的思雨，说真的她除了海拔低了点，其余什么都好；当然，我就是那个人见人爱的小仙女（不要迷恋我，我只是个传说！）哈哈，开玩笑啦。

  在军训接下来的几天里，我们可爱又可敬的教官，带我们一起度过了，白天军训脚一直离地的酸痛，手自然下垂的舒适，短暂休息时的开心；夜晚斗志昂扬的我们唱出军歌的高亢，乐此不疲的我们吼出斗歌的激亢，累并快乐着。
军训，令人恐惧，令人感激。恐惧它让我们亲身经历“魔鬼的训练”；感激它教会我们应迎难而上，披荆斩棘。大学生活，应充实，应奋斗，应疯狂，应无悔！最后，我想说：我很庆幸来到牧专，很庆幸被分到17兽医01，很庆幸遇见你们，大爱兽医01！

编辑人：李亚楠，李鹿也

活动日期：2017年9月11—22

入学感想-邓智鸿

发表于 2018-01-29 | 分类于 那些想说的话

  转眼之间，已经上了大学。经过了半个学期的学习我发现真正的大学和我想象中的有些不一样，人们都说高中好好学习吧，到了大学你就解脱了。到现在我才明白原来我们从高中这个火坑跳到了另外一个火坑里。
  曾经我认为大学是一个谈情说爱的地方。那有没有教导处的监管，没有大量的课后作业，也没有成摞成摞的试卷。有的只是丰富的课余时光，合得来的室友，关系融洽的小伙伴，各种有趣的活动，美丽的校园景色以及成对成对甜蜜的情侣。有课余时间就在图书馆里翻找资料，和对象一起在学校附近有名的地方游玩，或者和室友时不时的一起去KTV唱歌，去网吧开黑。做自己喜欢做的事，没有父母唠叨不休的教导，可以说是真正的放飞自我。

青春不负自己-孙亚敏

发表于 2018-01-29 | 分类于 那些想说的话

  在进入大学之前，老师和家长都会对我们说大学是自由的天堂。当时我对这句话是深信不疑的。想着只要考上了大学，我就再也不用每天背化学公式，写英语单词了，不用每天面临着数不尽的试题，每天做题做到十一二点了。于是就开始为考大学而努力的学习。我们高三的班主任是一个刚毕业的大学生，没有其他资深的老师那么有经验。为了让我们更加的努力学习，就时不时的跟我们讲她上大学时的一些趣事，她告诉我们：她在洛阳上的大学，洛阳有名的地方她都去腻了，周末的时候和朋友逛逛商场啊，买买衣服啊，到其他城市找老同学玩啊，每天两三节课，业余时间很多，总之，上了大学特别轻松，特别自由。确实每次听她讲后，我都特别想尽快的进入大学。十二年的寒窗苦读，十二载的披荆斩棘，承载着太多的梦想和追求，欢乐与痛苦，经历过千军万马过独木桥的汹涌之势，怀着期盼的心，踏着希望的音符，大家来到这个梦想起飞的地方“河南牧业经济学院”。现在我们已经经过了大一的一个学期，此时此刻不知道大家的心情是否和我一样，有了半年的大学生活，我们都相信了：命运并非上天的安排，命运是掌握在自己的手中，要获得幸福，拥有辉煌，就得去耕耘去锻造，就得左右自己。

入学感想-李亚楠

发表于 2018-01-28 | 分类于 那些想说的话

  在丹桂飘香、丰收的金秋五月，我怀揣梦想，满怀期待地踏入了大学校园。一段崭新的学习生活即将开始。
  记得刚踏进校园时，我拎着大包小包，迷茫地走着，摸不清方向，是热心的学姐、学长们对我关怀备至。他们热情地从我手中拿走包裹，帮我提着，带我报道、送我到宿舍。突然就觉得心里有一股暖流在涌动着，他们让我看到了同学之间那种真挚又可贵的爱。
红彤彤的烈日下，我迎来了期待已久的大学军训：挥洒汗水，斗志昂扬，我带着流血流汗不流泪的心，度过了十二个旭日东升和夕阳西下。军训留给我的是坚定的毅力，是严于律己，树立规范意识，是面对挫折，苦难咬牙坚持，迎难而上。
