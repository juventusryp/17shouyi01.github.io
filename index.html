暖冬行动-郑州福利院

发表于 2018-04-04 | 分类于 班级活动

  金秋已过，初冬已至。转眼间进入十二月，万物因气候的变化而变得落寞，太阳似乎也被冻的散发不出来热量，天气虽冷，但人心不冷，应学校要求，我们举行了志愿者活动—探望儿童福利院。

  当天一早，每个人已经准备就绪，经过了漫长的车程，我们到了郑州市儿童福利院，福利院中的孩子要么是因先天性疾病父母遗弃的，要么是属于留守儿童或是双亲去世的可怜儿，和他们相比，我们真的很幸运，也很幸福。

  福利院中的班级是按照年纪与智力的不同而划分的，我问工作人员这都在教些什么，工作人员说在教一些生活必备技能以及社会的认知，福利院的孩子们很少能出来看看世界，偶尔能够出来旅游，所以在他们眼中对社会的便民设施还是很陌生，其中我印象最深的就是一个孩子坐着遥控轮椅，不小心撞到了我，他立刻用流顺的英语对我说了“I’m sorry。”我明显的看出来他的智力是有问题的，但是他无意碰到了我能用英语给我道歉我却是真没想到。

  此次活动受到了福利院老师的热切感谢和赞扬。我班并与福利院达成经常进行帮扶活动的意愿。在看过孩子们在福利院的生活、学习、医疗环境后，志愿者全体成员都感到非常欣慰。

  这次活动让我班志愿者们认识到身边还有一些需要我们关注和伸出友谊之手的弱势群体，在日后的学习生活中除了规划自己的人生目标，更有责任去关注社会。

编辑人：王瑞翔 邓智鸿

龙福山庄游记

发表于 2018-04-04 | 分类于 班级活动

  茫茫人海中，相遇即是一种缘分。美好的回忆使我们的生活更富有诗意，深刻的道理引领我们走向正确的方向，感谢企业班主任的用心安排，游览龙福山庄之后，我们每个人心中都有了一份美好。

  看到龙福山庄大门虽很平常，但深刻感受到了
那里的农家气息，孩童的欢乐，同学们的畅所欲
言，瞬间这热闹的场景，让我们即将开始的活动
如虎添翼。活动是以分队为开始，然后我们就开
始了游戏――五毛一块，游戏开始的我们既激动又
紧张，随着游戏的逐渐进行，我们都玩的乐此不疲，超级开心，哈哈哈哈……

  之后就进行了限时游戏――自创游戏。在这过程中
各小组战队从开始的焦急、无规律到最后的从容
、有方法，这中间大家都体会到了企业班主任的
用心，游戏结束后大多同学都体会到了这点：团
结就是力量、时刻保持清醒才是硬道理。

  之后就又进行了限时游戏――互传呼啦圈。在上两
个游戏以后各战队队员之间的默契度逐渐提高,所以在这个游戏中同学们都在互找提速的方法，都在细心的去找缺点，当时各小组的凝聚力相信就算‘天崩地裂’也不会被分散开。这效果归功于
我们企业班主任，更是有了同学们之间的相互沟通，相互合作，甚至一个个聪慧的大脑换来的！最后赢者们赢的理所当然，败者也虽败尤荣。这样的结局不管何时都是令同学们满心欢喜的。企
业班主任虽说是以这一天积分来决出胜负，但大家只是为了给自己小组争光，并没有说要得到什么，这样无目的纯粹的去投入到游戏才是玩游戏的最高境界，当然也是之后工作中，生活中做事
的最佳方式吧。接下来大家都带着一身的喜悦和互相嬉闹着走进那附有农家气息的饭堂，虽然那里的饭菜没有学校的好吃，但大家因累的原因也是狼吞虎咽，那脸上满足感是在学校从来没有过
的。这让我们大家又一次深切体会“锄禾日当午，汗滴禾下土”的艰辛！生活本就是酸甜苦辣，不可能每天都活的有滋有味。这仿佛又在大家味味俱到的调料中增添了一点苦涩，让大家懂得了
珍惜粮食的重要性。下午企业班主任有组织大家自行组队去游览龙福山庄的一部分――千稼集。千傢集以“乡土的味道、年轮的烙印”为文化核心，还原经典历史场景再现民国初期到文革时段这六
十年的农村文化主题景区，到处是拉着伟大毛主席的语录条幅，真的真的很具有时代感，真的感谢企业班主任让我们体验一番。同时同学们的相互娱乐相互打趣那种欢乐的感觉仍记忆犹新。

  最后要回学校的时候，大家都带着企业班主任
的奖励和临近平安夜送的苹果照了象征结束一天的合照，照片中大家都如孩童般的微笑，更是对未来有了更多的憧憬！就这样结束了，在坐上回学校车的同时，大家心中更应清楚的是自己的今
后。在以后的社会和生活中“没有规矩不成方圆”希望以后的我们都能够有效的得到自我历练和提升，逐渐走向更成熟更优秀的自己。既然选择了远方，便只顾风雨兼程。一起加油吧，亲爱的同
学们！

青马课素质拓展

发表于 2018-04-04 | 分类于 班级活动

  随着最后一项报数活动的结束，我院青马工程培训落下帷幕。 作为参加到这次培训活动的学员, 我感到十分荣幸。

  青年时期是一个可塑性很强的时期，但是往往有许多潜能，被自己忽略或否定。比如在为人处世方面缺乏自信心等。而青马课的目的，就是让我们知道作为当代大学生，当代的共青团员，应该有充分的自信心去完成领导分配下来的工作和任务。同时让我们了解到，作为学生干部的我们应当以身作则，做一只高飞的领头雁。

  此次培训活动内容丰富多彩，不仅有不同老师专讲的七节精彩的课程，还有户外素质拓 展训练等活动。使我们在每周的学习中都能感受到学校团委对此项工程的重视以及该活动对我们的意义。虽然此次活动是题为“青年马克思主义者培养工程”，意在强调对青年马克思主义者的培养，然而我感觉我们学到的不仅是枯燥的理论知识, 反而更多的是我们学生干部的交际能力以及提高自我的素质。

  我院张书记以“做一只高飞的领头雁”为主题的课程，拉开了青马课的序幕，张书记运用精彩的演讲，使我们明白了自身的位置，“学生干部≠官”，“学生干部是一个自愿服务工作”，这些直白却震撼的语句，使我牢牢铭记在心里。

  短短的七堂课程里，既有结合我们大学生实际的做人、做事的道理，也有做为学生干部应掌握的技能方法，还有我们容易忽略的社交礼仪等。 这些课程使我消除了原本对参加此次培训的消极情绪，反而渐渐地喜欢上了该活，也更加珍惜得到此次培训的机会。青马课的培训，使我在很多方面都有了提高，例如应该怎样摆正学习的位置，如何与人交往，沟通的技巧，怎样保持乐观向上的心态等。这些不仅改变了我现在对自己、周围的许多看法，同时也将在今后对我起到重要影响。

  当然，使我印象深刻的不仅仅是青马课的精彩的课程，还有室外的素质拓展训练。在为期两天的拓展训练里，发生了许多震撼了我的内心的事情。

  在拓展训练里，每个队有一名队长（男），一名政委（女）。队员犯错，队长跟着一起承担受罚，看似不公平的规则里，却让我们明白了身为学生干部的我们，何尝不是这样一种身份的存在，我们的责任感和担当，尤其的重要。在活动中，有一个小队因为比赛总是在后几名，所以队长和政委一直被惩罚，队长已经不堪负重，终于，一个队员大声反抗了起来，却不曾想，给队长带来了更大的惩罚。当时的画面，感动了许多参加此次活动的人，有的人满含眼泪，有的人不忍看，悄悄的背过身去，当时的我们，内心被一种叫做不公平、凭什么的愤恨和懊恼却无能为力的感觉充斥着。但冷静后的我们，再次回想起这么一幕的时候，内心更多的是成就感和应有的担当责任感。素质拓展训练中，我担任的是我们队的政委，当最后一个报数活动结束后，看着我的队员们相拥而上来搀扶着我的时候，我惊奇的发现我的内心再也不是抱怨和怨恨，而是自豪感和担当责任感。

  作为一名学生干部，一方面是我们在工作中会遇到各种各样的问题，干什么事都不是一帆风顺的，在此时我们就必须学会控制自己的情绪，不能因为自己的情绪影响到工作。6另一方面是作为学生干部，我们并没有什么特权，我们的职责就是为同学服务。所以在工作之中，我们必须设身处地的为同学们着想，时刻从广大同学们的利益出发。还有一点就是要不断的进行自我反省，反省自己在工作中存在的问题，虚心接受同学指出自己的不足之处，用两面性的观点分析问题，不断提高为人处事的能力。

  作为学生干部我们应该有正确的思想动机和坦荡的胸怀，因为工作的缘故，学生干部工作无疑会使自己可支配的时间大大减少了。所以我们只有提高自己的组织协调能力，才能有激情的投入工作，不去追究它到底能给自己带来什么样的好处，从而摒弃了一些错误的思想观念。当工作与学习或者别的什么事相互冲突时，我们可以寻找一个平衡点，不要轻易放弃工作。在工作之中或许会与同事、同学之间产生摩擦，但是我们应该以坦荡的胸怀去面对，从而处理好两者之间的关系，才有利于处理好事情。同时必须要有高度的责任感和责任心。作为一名学生干部需要以“上情下达，下情上报”为原则、以“全心全意为同学服务”为宗旨开展工作，处理好与上级、同事、同学之间的关系。因此提高自身高度的责任感是必不可少的。

  通过此次培训，我在思想上也发生了许多转变。我感觉到在大学生活里要正确的认识自我，了解自我，发现差距，努力缩小差距。有的人怕别人知道自己的缺点而刻意去隐瞒，这是自卑的表现，其实在大学里衡量学生的好坏不只是成绩，而是全方位的综合的评价。人不可能是十全十美的，我们要敢于面对自己的缺点。找出自己的缺点和不足，并通过不断地努力的克服这些缺点和不足, 努力把自己营造的更加完善. 要不断提升自己各方面的能力。作为学生干部，组织活动，开展活动，必须具备许多方面的能力，特别是分析问题能力，组织活动能力，语言表达能力以及人际交往能力。具备这些能力之后，我们就能更加从容的面对工作中存在的问题，也更能得心应手的处理这些问题。

  其次，因为这次参加培训的学生都是本院系的学生干部，因此这不仅给我提供了一个与各类优秀人的沟通的平台，还让我认识到自己的不足，学到一些有用的工作方法。这使得我每次所参加的活动不是单纯的活动，而是发现身边优秀的人的舞台。当听到他人内涵深刻的见解，我便感到自己在某些方面的差距是显而易见的。而自己应该在这方面多努力，提高自己与人沟通的能力，组织、领导能力。而我也在每次的接触中，认识了更多的人，增进了对彼此的了解，使自己融入到了一个团结和谐的集体中去。此次活动我们所结识的同学将会成为彼此日后宝贵的财富。因为每个人身上都有独特之处，这也是这个干部群体所独有的优势，我们可以取长补短，发挥每个人的优势，在今后的学习、生活、工作中相互借鉴、彼此帮助。无疑为每个人日后的发展增添了筹码。

  当然，此次培训活动带给了我很多益处，同时也使我清楚的认识到了自己的不足，使我明确了今后的努力方向。首先，坚定的学习马克思主义的基本理论的信念。这种信念不是单纯的学习书本上的枯燥理论，而是学会学以致用。用自己的理解与实际行动去诠释这一理论，使更多的青年学生一起加入到成为马克思主义者的队伍中来。其次，学习、工作要注意方式、方法，协调好二者的关系。要培养比较宽阔的视野，善于学习，灵活工作。多方面的掌握知识技能，为今后的生活、工作打下坚实基础。第三，有大局意识，广泛联系群众，培养与广大人民的深厚感情。学校、组织对自己的培养绝对不是盲目的，而自己要努力成为的不仅是一个合格的大学生，还应当是一名合格的学生干部，以及一名心系群众、与广大人民血肉相连有社会责任感、使命感的党员干部。只有坚定这样的意识，才不负组织对自己的培养。而自己也应该以严格的标准来要求自己。我相信，只有个人觉悟提高了，其行动才会更有目的性，才能取得预期的效果。

  学生干部是校园中最具特色的一个团体。我很有幸成为其中的一分子，也有幸能够参加本次的青年马克思培养工程，而在本次培训中我所获得的东西将使我终身都受益, 同时我也会不断改进自己的工作方式，并在以后不断充实自己，使自己成为一个真正服务同学的学生干部。虽然此次活动要告一段落，但是对马克思主义的学习还没有结束，如何成为一名合格的学生干部及青年马克思主义者还需要我付出更多的努力。因此，今后我将通过此次培训活动的相关经验不断激励鞭策自己，不辜负学校、组织对我的期望。

欢度元旦

发表于 2018-04-04 | 分类于 班级活动

  生活不止眼前的狼狈，还有美好的昨天和未知的明天。

  眨眼间已是春光明媚，又是一年新学期，新生活，新起点。而过往经历的种种似乎还在昨天。不仅是初入大学报道的场景，还有和朋友们一起度过的难忘的青马培训活动，以及犹在昨天一般的元旦晚会等……而其中最为难忘的大概是去年初入大学时举行的迎新晚会。

  作为一个有幸参加的成员，当初和学长学姐以及同学们相识的场景还历历在目。从在班级群里看到征集节目时的心动，到在军训时的心不在焉，再到海选时的忐忑不安，以及训练过节目时的疲惫难过，直至最后正式演出时的开心喜悦… 其中种种，令人难忘，无需与外人道 已然铭记于心。

  上台时的喜悦大概每一位有幸一起参加的同学都有同感。而这次活动带给大家的更不仅如此。
作为动物医学院的一员，单单只是观看这次活动就令人回味无穷。

  开场七位学姐的街舞带动了全场的气氛，真的是光芒璀璨，令人侧目，几乎打动了所有有幸欣赏的人。接下来的歌曲演唱等音乐节目也为活动增添了不少光彩。喜剧小品《哑剧》在排练时就让我们欢笑不断，上台时的效果甚至比大家预期时的更棒，令人捧腹。多人参加的走秀节目更是令人期待，古装秀为大家带来了汉服的表演，更是宣扬了我们国家自古流传的服饰——汉服，引起了很多人对此的共鸣。

  这次晚会不仅为大家带来了一场视觉上的盛宴，让大家对未来的大学生活充满了期待，更是无形中宣传了动物医学院的五大系队，精彩帅气的开场舞是由动医街舞队的学姐们所带来，令人回味无穷的古典舞《无归》是古典舞队的学姐所奉献，最大的亮点更是由表演队所带来的哑剧《较劲》，主持队和礼仪队的学长学姐们更是协助大家表演了走秀并成功完成了主持这一重任。而由此带来的效果也十分显著。还有我们的压轴节目，一个及身高与嗓音于一体的选手——常鹏昌。将《蓝莲花》唱出了另一个顶峰。系队招新时报名的同学几乎无一不是因迎新晚会时的精彩节目而心动参加的。

  而我由此得到的锻炼和得到的感受也十分令人难以忘怀。只有相信自己，才能做到更好。
通过参加这次难忘的活动，我不仅结识了兴趣相投的好友，更是锻炼了自己的综合能力。相信不少同学也都有这样的经历和感慨。

  回忆总是美好的，即便当时有哭有笑，酸甜苦辣都尝了一遍。时至今日，回忆过往种种也是甘甜为上。
  如今又是一年新春景，春芽悄然绽放，春风悄然拂过，明媚阳光之下，是崭新的笑脸。
明天的我们，必然更加美好，我们的明天，必然更加灿烂

军训

发表于 2018-04-03 | 分类于 班级活动

  军训，是我们大一新生进入大学的第一节课，也是大学中最严格的一节课。有的同学听到军训就会想到暴晒的阳光、黝黑的皮肤、苛刻的训练、还有黄教官那不怎么严肃的表情。的确，军训时期没有正常学习生活过的舒服，但它却让我们难得体会到了后备军或其他军官在壮大国防力量时所受的辛苦和劳累。况且我们做的那些训练对于教官们来说“so easy”，根本不算什么。

日本70岁退休谈起

发表于 2018-03-08 | 分类于 转载

2018年初，日本政府修改法律，推迟公务员的退休年龄。

入学感想-邓智鸿

发表于 2018-01-29 | 分类于 那些想说的话

  转眼之间，已经上了大学。经过了半个学期的学习我发现真正的大学和我想象中的有些不一样，人们都说高中好好学习吧，到了大学你就解脱了。到现在我才明白原来我们从高中这个火坑跳到了另外一个火坑里。
  曾经我认为大学是一个谈情说爱的地方。那有没有教导处的监管，没有大量的课后作业，也没有成摞成摞的试卷。有的只是丰富的课余时光，合得来的室友，关系融洽的小伙伴，各种有趣的活动，美丽的校园景色以及成对成对甜蜜的情侣。有课余时间就在图书馆里翻找资料，和对象一起在学校附近有名的地方游玩，或者和室友时不时的一起去KTV唱歌，去网吧开黑。做自己喜欢做的事，没有父母唠叨不休的教导，可以说是真正的放飞自我。
